## Deferred Features

### TODO-2: 暗色模式
- **What:** 支持暗色/亮色主题切换
- **Why:** QA 经常加班，暗色模式减少眼疲劳；Tailwind 实现成本低
- **Context:** Tailwind CSS `dark:` 前缀，shadcn/ui 内置支持。加一个 ThemeToggle 按钮
- **Effort:** S (human: ~1h / CC: ~20min)
- **Priority:** P3（视觉优化，功能稳定后）
- **Source:** CEO 审查扩展 #2

### TODO-3: Prompt 注入防御增强
- **What:** 增强输入安全层——清洗用户输入中的元指令和角色切换尝试
- **Why:** System Prompt + User Input 模式存在 prompt injection 风险
- **Context:** 当前 System Prompt 已有基础防护（"只分析需求内容"）。后续可加输入清洗层：检测并移除 "ignore previous instructions" 等模式
- **Effort:** S (human: ~2h / CC: ~30min)
- **Priority:** P2（中风险，用户量上来前做）
- **Source:** CEO 审查 Section 3
